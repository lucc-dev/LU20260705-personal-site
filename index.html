開發工作流 / Workflow

這是我開發後端 API 系統時，一貫堅持的「嚴謹開發 4 步驟」（點擊可查看我的思維）

STEP 01

容器化建置環境

利用 Docker-compose 一鍵拉起 MySQL 與 Redis，確保開發環境完全隔離與整潔。

STEP 02

商務邏輯與 RESTful

撰寫 Java 與 Spring Boot Web API。利用 JPA 或 JDBC 底層存取資料庫。

STEP 03

記憶體快取加速

整合 Redis，將高頻重複讀取的基礎資料（如分類）存入快取，大幅減輕 DB 負擔。

STEP 04

Postman 完整驗證

嚴格驗證 HTTP 狀態碼與 JSON 欄位結構，排除 Bug 後才交付程式碼。
